Week 01 - Introduction and Preliminaries

This is the main website for the course QF 465: C++ For Finance. We will be discussing a number of topics ranging from C++ necessary for job interviews, the tools necessary for derivative pricing models, and extending interpeted languages such as Python and R using C++. If time in the course permits we will discuss how to extend developed models using CUDA C++ to run on NVIDIA GPU's.
